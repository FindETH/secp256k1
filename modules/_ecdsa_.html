Const sign

  • sign(message: string, privateKey: Buffer, deterministic?: boolean, curve?: Curve): ECDSASignature
    • Defined in ecdsa.ts:106

    Sign a message with the private key. When deterministic is set to true (default), this will use RFC6979 to generate a deterministic k value, otherwise a random k value is used.

    Note that the message is automatically hashed and prefixed, to match the hashes used by eth_sign.

    Parameters

    • message: string
    • privateKey: Buffer
    • Default value deterministic: boolean = true
    • Default value curve: Curve = secp256k1

    Returns ECDSASignature
